ATTENDANCE
MANAGEMENT
SYSTEM




ABSTRACT

  1. Taking attendance manually is generally time consuming and cumbersome.

  2. Since this invokes the possibility of errors, there is need for a better data management system that is always quick and accurate.

  3. In this project we have designed an RFID & Fingerprint based Attendance System using Arduino, RFID MFRC522 Module and R307 fingerprint module.

  4. This application takes attendance electronically and the records of the attendance are stored in a database to which only the teacher has direct access.

  5. Students can mark their attendance using fingerprint. Teachers are issued an RFID card for admin access.

COMPONENTS USED

  1. ARDUINO UNO

  2. 4 X 4 MATRIX MEMBRANE KEYPAD

  3. 16 X 2 LCD DISPLAY

  4. R307 FINGERPRINT SENSOR MODULE

  5. MF RC522 RFID MODULE

  6. DS3231 RTC MODULE

  7. DOT / VEROBOARD

  8. TOGGLE SWITCH




RESULT

PHASE 1 implemented. PHASE 2 partially implemented. Few constraints noted and working on the rectification. Function to add and delete fingerprint data created. Started designing 3D case for the prototype using fusion 360.
